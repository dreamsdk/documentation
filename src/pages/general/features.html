Features

DreamSDK is a very different environment for the Sega Dreamcast development in comparison of previous released packages of this kind:

  • Fast and easy to install: Just double-click on the setup file and let the program install & configure everything for you.
  • Lightweight: Thanks to the MinGW/MSYS environment, this environment is really smaller in disk size instead of competitors solutions like Cygwin (the other alternative for UNIX® like environment on Windows).
  • Ready to use: All the toolchains (which are really long to build, almost 8 hours!) are already prebuilt and ready-to-use.
  • Configurable and upgradable: With the DreamSDK Manager tool, install or update the KallistiOS (KOS) library really easily. Handle KallistiOS Ports through that user-friendly application, and configure Dreamcast Tool to fit your needs.
  • Respect of the KallistiOS standards configuration and documentation: This environment is 100% compliant with the KOS standards and documentation. No major specific hacks or dirty workaround have been done (the only exception is the ln tool, which is not working properly on MinGW/MSYS but this is a really minor drawback and it's already handled by DreamSDK Manager).

DreamSDK is a nice way to develop for the Sega Dreamcast on Windows!
