Utilities and helpers command-line tools

This sections details all utilities and helpers command line tools available into DreamSDK.

These tools are installed in the /opt/dreamsdk/helpers and /opt/dreamsdk/scripts directories.

Please select a tool below to learn more.

Utilities:

  • Browse — Current directory opener in the Windows® Explorer (browse)
  • Dreamcast Tool — Dreamcast sender/receiver (dc-tool)
  • ELF to BIN — Dreamcast Executable Linkable Format (ELF) to RAW Binary (BIN) converter (elf2bin)
  • Make Disc — Selfboot disc generator (makedisc)
  • MinGW Get Information Tool — MinGW Packages Information Lister (mingw-get-info)

Helpers:

  • DirHash — Directory hash generator (dirhash)
  • DreamSDK Get Info — DreamSDK information retriever (dreamsdk-get-info)
  • FastARP — Fast Address Resolution Protocol utility (fastarp)
  • FastPing — Fast Ping utility (fastping)
  • IMG4DC — Dreamcast Selfboot Toolkit
    • CDI4DC — Padus DiscJuggler image generator (cdi4dc)
    • MDS4DC — Alcohol 120% image generator (mds4dc, lbacalc)
  • IP.BIN Reader — Bootstrap Fields Reader (ipreader)
  • Make ISO File System — ISO9660 image generator (mkisofs)
  • UAC Status — User Account Control Status (uac-status)

Utilities

Browse — Current directory opener in the Windows® Explorer (browse)

The Browse simple utility was made to quickly open the current directory from your DreamSDK Shell into the Windows® Explorer.

In clear, if you are in the ~/myproject directory in the DreamSDK Shelland you enter the browse command from that directory, the %DREAMSDK_HOME%\msys\1.0\home\%UserName%\myproject directory will be shown in the Explorer.

This may be useful for daily usages.

Dreamcast Tool — Dreamcast sender/receiver (dc-tool)

Dreamcast Tool is an Ethernet/Serial loader for the Sega Dreamcast. This program is part of KallistiOS. The dc-tool script included in DreamSDK is a wrapper for the real dc-tool-ip and dc-tool-ser binaries built when installing KallistiOS from DreamSDK Manager.

This simple wrapper uses the connection and options settings defined in the DreamSDK Manager tool so you don't have to pass them all the time to that script. You only have to pass the command you want (e.g. -x, -d...) and the target file name.

For example, if you want to execute a program, you just have to enter the following: dc-tool -x myprog.elf, instead of entering all the connection/option settings, like dc-tool -x myprog.elf -t COM1 -b 115200. The -t and -b switches are automatically passed to the real dc-tool program (i.e. the dc-tool-ip or dc-tool-ser binary) from the choice you made in the DreamSDK Manager.

Please note, this tool will need the dcload disc. Read more here.

ELF to BIN — Dreamcast Executable Linkable Format (ELF) to RAW Binary (BIN) converter (elf2bin)

ELF to BIN is a simple shell script that will convert a Sega Dreamcast ELF program to the RAW Binary (BIN) format. This will uses the kos-objcopy wrapper in the background.

After doing this conversion, you will have to scramble that BIN program in order to create a bootable disc.

Read more about this topic here.

Make Disc — Selfboot disc generator (makedisc)

The Make Disc script is a very important tool made for creating selfboot images of your programs. It will requires the usage of makeip (part of KallistiOS), ELF2BIN and IMG4DC. Read more here.

MinGW Get Information Tool — MinGW Packages Information Lister (mingw-get-info)

This small tool may be useful as it prints the whole list of the MinGW packages currently installed.

Please note, you may use the MinGW-Get tool in order to update these MinGW packages.

Helpers

All these helpers exists because they are used by DreamSDK itself but they may be useful for you, that's why you'll find the description of each of these components in that section.

DirHash — Directory hash generator (dirhash)

The DirHash utility is a great tool to compute a global hash for a directory. It's widely used in the DreamSDK environment for various processes but it may be useful for you too.

DreamSDK Get Info — DreamSDK information retriever (dreamsdk-get-info)

This simple tool was made to retrieve some information on the DreamSDK environment you are currently using:

  • Pass the --release switch to get the version number information;
  • Pass the --date switch to get the build date.

Only one switch can be passed to the command-line at one time.

FastARP — Fast Address Resolution Protocol (ARP) utility (fastarp)

FastARP is a simple wrapper used to assign an IP address to a MAC address. This tool is used in the DreamSDK engine but may be useful for you. It uses arp or netsh in the background, depending on your OS.

FastPing — Fast Ping utility (fastping)

FastPing is another simple wrapper used in the DreamSDK engine. It replaces the ping tool by something really faster than the regular ping.

It may be useful in some batches.

IMG4DC — Dreamcast Selfboot Toolkit

IMG4DC is the core engine tool used in the makedisc script. IMG4DC is a package that allows you to generate bootable CD images for your Sega Dreamcast. It's split in 2 separate programs, usable in command-line mode:

  • CDI4DC : for generating Padus DiscJuggler CD images;
  • MDS4DC : for generating Alcohol 120% CD images.

These programs have become since 2007 a reference in the Sega Dreamcast homebrew world, and they are used in other third-party tools such BootDreams.

CDI4DC — Padus DiscJuggler image generator (cdi4dc)

CDI4DC generates Padus DiscJuggler CD images, which is the standard for the distribution of Sega Dreamcast programs.

This tool generates cdi files.

MDS4DC — Alcohol 120% image generator (mds4dc, lbacalc)

MDS4DC generates Alcohol 120% CD images. It's another format used for the distribution of Sega Dreamcast programs.

This tool generates mds/mdf files.

IP.BIN Reader — Bootstrap Fields Reader (ipreader)

The IP.BIN Reader tool is a small utility program which will read the fields of a Sega Dreamcast bootsector, usually known as the IP.BIN file. This tool is used internally but it may be useful for you too.

Make ISO File System — ISO9660 image generator (mkisofs)

Make ISO File System is an utility that creates an ISO 9660 image from files on disk. This tool is used internally by the DreamSDK environment package.

UAC Status — User Account Control Status (uac-status)

UAC Status is a helper program made to determine if the User Account Control (UAC) is enabled on the system. On systems which doesn't support UAC it will indicate the disabled status.
