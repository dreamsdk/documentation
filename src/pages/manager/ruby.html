Ruby

This tab is here to manage the Ruby library for the Sega Dreamcast. This uses mruby implementation. As almost all Sega Dreamcast open source software, this uses KallistiOS (KOS) as base.

This feature is experimental.

You have the possibility to activate this feature by clicking on the Install… button.

After the installation, you can Update the library using the appropriate button (of course if you have selected the online package).

The Start Shell… button will help you to open your DreamSDK Shell directly in the directory where the samples are stored. The Open Samples… button will just open the samples directory in Windows® Explorer.
