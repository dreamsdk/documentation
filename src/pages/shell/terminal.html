Terminal

With the DreamSDK Shell, you have the the possibility to choose the Shell interface of your choice, between:

  • Windows Prompt (by default)
  • MinTTY

Click here to learn how to configure this Shell interface.

Command Prompt

The default Shell interface usable with MSYS is the standard Windows Command Prompt (cmd).

In that case, you will be able to use the DreamSDK Shell (i.e. the MSYS Shell) in the Windows Command Prompt window like a standard Command Prompt. But remember, despite of the UI appereance, you are in a Bash shell!

MinTTY

Another Shell interface alternative usable with MSYS is MinTTY.

For your information, this Shell is now the standard Shell for the Cygwin and the MinGW-w64/MSYS2 projects.

MinTTY is a free and open source terminal emulator for Cygwin, the Unix-like environment for Windows, but it supplied with the MSYS team too, that's why this option is available for the DreamSDK package.
