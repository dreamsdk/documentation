Changes Log

This section describes all releases and notable changes made to the DreamSDK environment.

R2 — 2019-02-17

  • Full Code::Blocks 17.12 integration!
  • GDB for SuperH is now the 8.2 release.
  • DreamSDK Runner is now part of the package. This application allows executions of instructions directly in the MSYS Shell but from the standard Microsoft Windows Prompt.
  • DreamSDK Shell has been simplified: it's now a MSYS Shell shortcut only (previous role is now handled by DreamSDK Runner).
  • DreamSDK Manager was completely redesigned.

First public release.
