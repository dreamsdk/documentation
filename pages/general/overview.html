DreamSDK Overview

DreamSDK is a ready-to-use environment for Sega Dreamcast development running on the Microsoft Windows platform.

It's a self-contained package embedding all needed software to freely develop for the Sega Dreamcast system on Windows. This package is built on open-source software only thus it doesn't contains any copyrighted software. See licenses for more information.

DreamSDK is composed by several components, mainly:

  • MinGW/MSYS for the core system and foundation;
  • Binutils, GNU Compiler Collection (GCC), Newlib and GNU Debugger (GDB) for all the toolchains;
  • Additional components made specially for this package:
    • DreamSDK Manager, which handles the KallistiOS library;
    • DreamSDK Shell, a component which is built on the top of MSYS.

DreamSDK uses KallistiOS (KOS), which is the best option today to develop for the Dreamcast. It's a pro-grade development system for the Dreamcast. KOS has been used to produce multiple commercial quality games for sale like Feet of Fury.

Instead of the majority of the previous released packages of this kind (e.g. dcfreedev — with GCC 3.4.6 —, DC Dev Environment (DDE), Code::Blocks DC Development...), this package only includes the pre-compiled toolchains (both for the SuperH CPU and Yamaha AICA) but not the KallistiOS library itself: KallistiOS is handled by the DreamSDK Manager application.

The whole package is built and optimized for Windows. It respects the KallistiOS requirements and prerequisites, which means that you'll need to compile your Dreamcast programs from the DreamSDK Shell.

This environment should work with Libronin too, the only other viable alternative for the Dreamcast development but this scenario was not tested.
