Features

  • Easy to install: Just double-click on the setup file and let the program configure everything for you.
  • Lightweight: Thanks to the MinGW/MSYS environment, the environment is really smaller in size instead of Cygwin.
  • Fast: All the tool-chains, which are really long to build (about 8 hours!) are already pre-compiled and ready-to-use.
  • Configurable and upgradable: With the DreamSDK Manager tool, install or update the KallistiOS library really easily. Handles KallistiOS Ports through an user-friendly application, and configure Dreamcast Tool at the same time.
  • IDE Compatible: As all tool-chains executables were built statically, they can be run outside the MinGW/MSYS environment so they can be used on IDE environments.
