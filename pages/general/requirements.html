Requirements

This section explains the requirements needed for DreamSDK.

  • Required external components
  • Required operating system
  • Internet connection

Required external components

DreamSDK needs the following external components to work properly:

  • Git
  • Python 2.7.x (32-bit)
  • Subversion Client (SVN)

Theses components are not embedded in the DreamSDK package thus they need to be installed separately.

All of these components should be available in the PATH environment variable. You can verify that by following this procedure:

  1. Open a Command Prompt (Hit the Win + R keyboard shortcut, enter cmd and hit Enter);
  2. Enter the following commands:
    • git --version
    • python --version
    • svn --version
  3. All of these commands should print the version of each components.

If at least one of them prints an incorrect message like Bad command or filename, then you should (re)install the problematic component, and please don't forget to check the PATH option when asked!

These components are absolutely required, so if they are not installed, DreamSDK will refuses to be installed.

Please also note, for Python, only the 2.7.x 32-bit version is compatible, as it's statically linked to GNU Debugger (GDB).

Required operating system

For the Windows Operating System (OS), Windows XP Service Pack 3 or later is required.

If you want to install DreamSDK on Windows XP, please pay attention to the external components version to use. The Windows XP support was dropped almost everywhere, the reason why DreamSDK support it today is the fact that the Sega Dreamcast is a legacy system released back in 1998; some of useful utilities were made for this operating system (or more exactly, for Windows NT which is older!).

So if you want to install DreamSDK on Windows XP Service Pack 3, please use these components versions:

  • Git: 2.10.0 and optionally TortoiseGit 1.8.16.0;
  • Python: any of 2.7.x release, e.g. 2.7.15 is compatible;
  • Subversion Client (SVN): 1.7.19 from CollabNet.

In my case, I have a workstation dedicated to the Sega Dreamcast environment running Windows XP. This explains also the choice of using the legacy MinGW/MSYS environment instead of the modern (and better) MinGW-w64/MSYS2 environment, as they officially dropped Windows XP support.

Of course, you can use DreamSDK on modern Windows, if you plan to develop for your Dreamcast and execute your programs on emulators or even use Dreamcast Tool. This software package is compatible with Windows 10.

Internet connection

In order to install the DreamSDK environment on your computer, you need an active Internet connection.

This is required as the KallistiOS library will be downloaded at the installation time.

If you don't have any Internet connection, you will have to install KallistiOS manually, but in this case, the Update feature of the DreamSDK Manager will be impossible. Sorry, this possibility is not documented at this time.
