Environment

DreamSDK uses the MinGW/MSYS environment provided by MinGW.org.

Toolchains

The DreamSDK package provides two toolchains, one for the Hitachi SuperH (SH-4) CPU and the other for the Yamaha AICA Super Intelligent Sound Processor (ARM).

Toolchains are built upon the following components:

  • Binutils
  • GNU Compiler Collection (GCC)
  • Newlib: http://sourceware.org/newlib/
  • GNU Debugger (GDB): https://www.gnu.org/software/gdb/

Libraries

The libraries provided by the DreamSDK package used for the Sega Dreamcast development are KallistiOS and its addons which are KallistiOS Ports and Dreamcast Tool. The official website of KallistiOS is http://cadcdev.sourceforge.net/softprj/kos/.

KallistiOS license is the specific KOS License, when all of the additional components are licensed separately.

Note: These libraries aren't directly embedded in this package but they are downloaded/compiled in real-time by the DreamSDK Manager tool, which is on the GNU GPL 3 license (see Miscellaneous section below for more information).

Dependencies

In order to use the DreamSDK package, the following dependencies are used:

  • Git: https://git-scm.com/
  • Subversion (SVN): http://subversion.apache.org/
  • Python: https://www.python.org/

Miscellaneous

The DreamSDK package embed two additional applications made by The DreamSDK Team:

  • Shell: The Shell Launcher component, which launch the MSYS Shell.
  • Manager: The Manager component, which administrate the DreamSDK installation.

All of these components were built with Free Pascal (https://www.freepascal.org/) and Lazarus (https://www.lazarus-ide.org/) and are licensed under the GNU GPL 3 license.

The Setup was made with Inno Setup (http://www.jrsoftware.org/).
