Environment

DreamSDK uses the MinGW/MSYS environment provided by MinGW.org.

This environment provides an UNIX® like environment on Windows.

Toolchains

The DreamSDK package provides two toolchains, one for the Hitachi SuperH (SH-4) CPU and the other for the Yamaha AICA Super Intelligent Sound Processor (which uses an ARM core).

Toolchains are built upon the following components:

  • Binutils
  • GNU Compiler Collection (GCC)
  • Newlib
  • GNU Debugger (GDB)

Libraries

The libraries provided by the DreamSDK package used for the Sega Dreamcast development are KallistiOSand the KallistiOS Ports and Dreamcast Tool addons. Click here to visit the official website of KallistiOS.

KallistiOS license is the specific KOS License, when all of the additional components are licensed separately.

Note: These libraries aren't directly embedded in this package but they are downloaded/compiled in real-time by the DreamSDK Manager tool, which is on the GNU GPL 3 license (see Miscellaneous section below for more information).

Dependencies

In order to use the DreamSDK package, the following dependencies are used:

  • Git
  • Subversion (SVN)
  • Python

Miscellaneous

The DreamSDK package provides two additional applications made by The DreamSDK Team:

  • Shell: The Shell Launcher component which handles the MSYS Shell.
  • Manager: The Manager component, which administrate the DreamSDK installation.

All of these components were built with Free Pascal and Lazarus and are licensed under the GNU GPL 3 license.

The setup installer was made with Inno Setup.
