Dreamcast Tool

The Dreamcast Tool is the tool responsible of sending your program to the Sega Dreamcast, in order to execute/debug it. This tab handles the configuration of this tool.

Please choose the configuration accordingly to your scenario:

  • If you have a Serial (Coders Cable), you should check the Serial button.
  • If you have a Broadband or LAN Adapter (which is a Ethernet card for the Sega Dreamcast), you should check the Internet Protocol (IP) button.
  • Or you can click the None button to disable Dreamcast Tool.

After choosing your tool version, the settings panel will activate. The description of the options are described in the KallistiOS documentation.

Please note, these options are applied all the time, every time you will run the dc-tool program.

At the Shell prompt, you just have to run the following command:

dc-tool -x <yourprog.elf>

Or, most of the time, this is exactly the same command as:

make run

This will start Dreamcast Tool with the settings you have set to run/debug <yourprog.elf>!

Please note, you have the possibility to run GDB to debug your program, please read the KOS manual about this.
