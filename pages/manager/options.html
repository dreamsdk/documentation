Options

The Options tab display all the repositories URL used by DreamSDK Manager to download the KallistiOS components, plus the Terminal choice, between the Windows Prompt or MinTTY for the DreamSDK Shell.

Note: If you already have installed KallistiOS, then the URL were not be used anymore. They are used only one time when cloning all the repositories..

You have also the possibility to run the original MinGW Installation Manager, if you want to add some features to the MinGW/MSYS environment used by your DreamSDK environment. Please also note, all the required MinGW/MSYS packages are already installed.
