Additional user-interface tools

This sections details all additional and optional user-interface tools available when installing DreamSDK.

All these tools are installed in the /opt/dreamsdk/tools directory.

Please select a tool below to learn more:

  • 1ST_READ.BIN Checker — Binary program state checker
  • BootDreams — Selfboot image generator frontend
  • IP.BIN Writer — Initial Program manipulation tool
    • Additional ready-to-use Initial Program (IP) logos
  • MR Writer — Initial Program picture format management tool
  • SBI Builder — Selfboot Inducer package generator
  • Selfboot Inducer — DreamInducer disc generator
  • VMU Tool PC — Visual Memory data manager (GUI)

1ST_READ.BIN Checker — Binary program state checker

1ST_READ.BIN Checker is a tool which help you to see if a Sega Dreamcast program is in scrambled or unscrambled state.

Scrambled or unscrambled states are described here.

BootDreams — Selfboot image generator frontend

BootDreams is a tool for Windows that uses many command-line tools to create selfbooting images and discs for Sega Dreamcast.

This is the MakeDisc utility equivalent but with a nice user-interface.

IP.BIN Writer — Initial Program manipulation tool

IP.BIN Writer is made to generate bootstrap IP.BIN (Initial Program) files for your Sega Dreamcast console. This tool is used to generate customized bootstraps (e.g. with custom logos) for homebrew programs only, mainly KallistiOS programs (but it should be compatible with libronin programs too).

This is the IP creator (makeip) utility (which is part of KallistiOS) equivalent but with a nice user-interface.

The bootstrap is the license screen shown at startup when you run any Dreamcast program.

If you wish to create a custom logo, it can be in BMP, PNG, GIF or JPEG format, with 320 × 90 in size, and with 128 colors max in the palette. The custom logo is in a custom format called MR, so using a BMP, PNG, GIF or JPEG format will convert the file in that MR format. The transparent color is #c0c0c0 (R: 192, G: 192, B: 192).

Additional ready-to-use Initial Program (IP) logos

You have the possibility to install some ready-to-use IP.BIN logos at the same time with IP.BIN Writer.

Logos will be available in /opt/dreamsdk/tools/ipwriter/iplogos/ directory.

MR Writer — Initial Program picture format management tool

MR Writer is a tool to generate IP.BIN logos for inserting in IP.BIN files. These kind of images are in the MR format, a special image format developed for the Sega Dreamcast.

Note: The IP creator (makeip) utility included in KallistiOS can handle MR image generation.

SBI Builder — Selfboot Inducer package generator

SBI Builder is a tool to build Selfboot Inducer packages (SBI) from a Sega Dreamcast program.

The SBI package is a method of distribution for your Dreamcast programs used with Selfboot Inducer.

Selfboot Inducer — DreamInducer disc generator

Selfboot Inducer is a program made to build a DreamInducer bootable compilation CD-ROM for your Sega Dreamcast.

DreamInducer is a Sega Dreamcast program allowing you to put multiple homebrew games, emulators, demos or applications onto the same CD.

To use this application, you must have the packages corresponding to the application you like, for example neo4all.sbi for the Neo 4 All emulator.

SBI packages can be generated by SBI Builder.

This application is meant to be used by the end-user which will receive your SBI package generated by SBI Builder. This application is provided in DreamSDK only for testing your SBI package.

VMU Tool PC — Visual Memory data manager (GUI)

VMU Tool PC is a graphical tool allowing you to manage all VMU/VMS (saves) files on your computer.

This utility is the user-interface of the VMU Tool PC command-line addon.
